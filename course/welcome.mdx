## How to add a chapter

1. Write a new `course/<slug>.mdx` file with the same frontmatter shape as this one.
2. Add one entry to `course/manifest.json`'s `chapters` array — `slug`, `title`, `summary`,
   `file`.
3. That's it — `pages/course-viewer.html` reads the manifest and renders the sidebar +
   the chapter content client-side. No build step.

## Markdown works as expected

Plain markdown — headings, **bold**, *italic*, `code`, lists, [links](https://recursive.eco),
and images — is rendered with [marked](https://marked.js.org/) client-side. Keep MDX-specific
syntax (JSX imports/exports) out of these files; this viewer renders plain markdown only, on
purpose, so there's no build step.
